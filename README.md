# lchighpass-web
## 项目插件
* element ui
* vue-quill-editor (富文本插件)
* less (css预处理器)
* eslint语法校验
## 项目介绍
* 本项目是一个用来帮助需要朗思考试的学生来对自己所掌握的知识进行测试
## 功能描述
* 本项目主要分为四大块: 听 读 写。
- 听：用户可以通过听取录音内容来进行答题
- 读：用户可以通过看提供的短文来进行答题
- 写：用户可以通过提示来进行作文的编写
